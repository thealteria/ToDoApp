# ToDoApp

ToDoApp is an open-source android application made using [Flutter](https://flutter.dev/) which allows user to save notes in a very easy way.

## Features 

- Add Notes
- Update Notes
- Delete Notes

## Development Environment

- Visual Studio Code v1.33.0
- Flutter SDK v1.2.2
